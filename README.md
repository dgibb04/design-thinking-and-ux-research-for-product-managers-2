## Audience Summary
- **Role:** Mid-level Product Managers

- **Goal:** Integrate structured design thinking into product sprints [Source: tmplehcb_af.pdf]

## Learning Outcomes
1. Explain the five stages of the Design Thinking process.
2. Write a research plan and conduct a structured user interview.
3. Use affinity mapping to synthesise qualitative research findings.
4. Write a clear How Might We (HMW) problem statement.
5. Build and test a low-fidelity prototype.
6. Integrate Design Thinking into product sprints [Source: tmpv88a4zsl.pdf].
